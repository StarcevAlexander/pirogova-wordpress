# Pirogova — Инструкция по развёртыванию

## Требования

| Компонент | Версия |
|-----------|--------|
| WordPress | 6.4+ |
| PHP | 8.1+ |
| WooCommerce | 8.0+ |
| WooCommerce YooKassa | 3.4+ (онлайн-касса) |

---

## 1. Установка темы

1. Скопируйте папку `pirogova-theme/` в `wp-content/themes/`
2. Активируйте тему: **Внешний вид → Темы → Пирогова**

---

## 2. Обязательные плагины

### WooCommerce
- Установить через **Плагины → Добавить** → поиск «WooCommerce»

### WooCommerce YooKassa (онлайн-касса + онлайн-оплата)
- Скачать с официального сайта: [yookassa.ru/developers/tools/cms-modules/woocommerce](https://yookassa.ru/developers/tools/cms-modules/woocommerce)
- Установить `.zip` через **Плагины → Добавить → Загрузить плагин**
- Настройка: **WooCommerce → Настройки → Оплата → ЮKassa**
  - Вставить `shop_id` и `secret_key` из личного кабинета ЮKassa
  - Включить фискальные чеки (ФФД 1.2)
  - Система налогообложения: УСН (доходы) или ОСН — в зависимости от вашего бизнеса

---

## 3. Настройка WooCommerce

### Создание товаров-пирогов (6 штук)

1. **Товары → Добавить товар**
2. Тип товара: **Вариативный товар**
3. Добавить атрибут «Вес»:
   - Вкладка **Атрибуты** → Добавить → Название: `Вес`, Значения: `500 г | 750 г | 1000 г | 1500 г`
   - Галочка **Используется для вариаций**
4. Вкладка **Вариации** → Сгенерировать все вариации
5. Для каждой вариации задать: цену, изображение (фото конкретного размера), SKU
6. Загрузить основное фото товара (будет в каталоге)

### Настройки доставки
- **WooCommerce → Настройки → Доставка** → Добавить зону доставки с нужным городом
- Методы: фиксированная стоимость или бесплатная при заказе от N рублей

### Настройки налогов
- Если работаете на УСН без НДС: **WooCommerce → Настройки → Налоги** → выключить налоги
- Если ОСН: настроить ставку НДС 20%

## 4. Настройка темы (Кастомайзер)

**Внешний вид → Настроить** → раздел **«Контакты и настройки»**

| Поле | Описание |
|------|----------|
| Телефон | Отображается в шапке и футере |
| Email | В футере |
| Адрес | В разделе контактов |
| Ссылка на карту (iframe src) | Src-ссылка из Яндекс.Карт или 2GIS для iframe |
| ВКонтакте URL | Ссылка на группу ВК |
| Telegram URL | Ссылка на канал/бот |
| Слоган в футере | Подпись под логотипом в футере |
| Фоновое изображение Hero | Главное фото на первом экране |

---

## 5. Добавление упоминаний в СМИ

1. **СМИ и пресса → Добавить упоминание**
2. Заголовок — название издания (напр. «РБК», «Коммерсантъ»)
3. Поля:
   - Ссылка на статью
   - Цитата из статьи
   - Логотип издания (через медиабиблиотеку)
4. Порядок отображения регулируется полем «Порядок» в атрибутах записи

---

## 6. Статусы заказов

После установки темы в WooCommerce появятся новые статусы:

| Статус | Значение |
|--------|----------|
| Готовится | Заказ принят, пирог в процессе выпечки |
| Доставляется | Курьер везёт заказ |
| Доставлен | Заказ вручён клиенту |

Смена статуса: **WooCommerce → Заказы** → открыть заказ → изменить статус в правом сайдбаре.

---

## 7. Главная страница (одностраничник)

1. Создать страницу: **Страницы → Добавить** → Название любое, содержимое оставить пустым
2. **Настройки → Чтение** → «Главная страница отображает» → выбрать «Статическую страницу», выбрать созданную страницу
3. Тема автоматически использует шаблон `front-page.php`

---

## 8. Цветовая схема

CSS-переменные находятся в `assets/css/main.css` в блоке `:root`. Для смены цветов бренда отредактируйте:

```css
--clr-primary:       #7B4A2D;   /* основной тёмно-коричневый */
--clr-primary-light: #9E6040;
--clr-accent:        #C8860A;   /* золотой акцент */
--clr-bg:            #FFF8F0;   /* тёплый белый фон */
```

## 9. Структура файлов темы

```
pirogova-theme/
├── style.css                    # Заголовок темы
├── functions.php                # Подключение всех зависимостей
├── front-page.php               # Одностраничный шаблон
├── header.php / footer.php      # Шапка и подвал
├── inc/
│   ├── theme-setup.php          # Поддержка темы, кастомайзер
│   ├── woocommerce.php          # Настройки WooCommerce
│   ├── order-statuses.php       # Кастомные статусы доставки
│   └── ajax-handlers.php        # AJAX: попап, корзина
├── template-parts/
│   ├── section-hero.php         # Главный экран
│   ├── section-catalog.php      # Каталог пирогов
│   ├── section-how-it-works.php # Как это работает
│   ├── section-media.php        # О нас в СМИ
│   ├── section-contacts.php     # Контакты
│   └── product-popup.php        # Попап товара
├── woocommerce/
│   └── checkout/
│       └── thankyou.php         # Страница «Спасибо за заказ»
└── assets/
    ├── css/main.css             # Все стили
    ├── css/admin.css            # Стили для админки
    └── js/main.js               # Попап, слайдер, AJAX-корзина
```
